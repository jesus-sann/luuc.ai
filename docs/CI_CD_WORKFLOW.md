## Quick Commands Reference

```bash
# Local Development
npm run dev              # Start dev server
npm run test:watch       # Watch tests (recommended!)
npm run lint:fix         # Auto-fix linting

# Before Pushing
npm run ci               # Full CI simulation (2-3 min)

# Debugging
npm run test:ui          # Visual test UI
npm run test:coverage    # See what's tested

# Build Validation
npm run build            # Test production build
npm start                # Run production build

# Type Checking
npm run type-check       # Verify TypeScript
```

---

**Pro Tips:**

1. Keep `npm run test:watch` running in a terminal while coding
2. Run `npm run ci` before pushing to catch issues early
3. Review preview deployments before merging PRs
4. Monitor GitHub Actions tab for CI performance trends
5. Check coverage reports to find untested code paths
